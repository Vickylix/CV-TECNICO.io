# CV-TECNICO -- Victor Manuel Marquina Pilares
Este proyecto es un CV tecnido usando HTML Y CSS.

## Descripcion 
Se trata de una pagina web simple donde presento mi perfil profesional, tecnologias que se usar y proyectos propios.

## Tecnologias utilizadas
* HTML
* CSS
## Funcionalidades 
* Presentacion de perfil
* Lista de tecnologias que se usar y defender
* Lista de proyectos de mi autoria, donde muestro mis conocimientos con enlaces
* Informacion de contacto
